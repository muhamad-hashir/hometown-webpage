While learning full stack web development from Scrimba, I completed the HTML and CSS part of the course. This hometown webpage was the solo final task of that section.

I built a travel webpage for Peshawar, KPK — featuring the top activities, famous food, historic places and a personal guide section.

Features used:
- Flexbox for layout and alignment
- Background images with cover sizing and center positioning
- Border radius for rounded images and boxes
- CSS classes for styling individual elements
- Padding and margin for spacing
- Inline-block for fitting boxes tightly around text
- Google fonts or custom font styling

Built using pure HTML and CSS.
